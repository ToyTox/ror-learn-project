Внимание! Для создания нового проекта используйте версию Rails 6.х (не 7)!

Создайте новый репозиторий с проектом, создайте новую ветку. Ведите разработку этого и последующих заданий в новых ветках (каждое новое задание в своей ветке). Ответ на это и последующие задания предоставьте в виде ссылки на пулл реквест.
Добавьте в конфигурацию Rails приложения своё местоположение: часовой пояс (how to: https://riptutorial.com/ruby-on-rails/topic/3367/change-default-timezone) и язык (для этого в файле config/application.rb измените значение I18n.default_locale в нужное значение, примеры: https://guides.rubyonrails.org/i18n.html#configure-the-i18n-module).
Добавьте Initializer с версией приложения (используйте https://semver.org/). Для этого добавьте файл version.rb с нужным кодом в config/initilizers/
Добавьте статическую страницу /about с кратким описанием проекта, включая ссылку на сайт онлайн-школы Thinknetica (http://www.thinknetica.com)
Добавьте статическую страницу /about/author с информацией об авторе (включая ссылку на GitHub профиль)
Добавьте в README.md список сущностей (объектов реального мира) для проекта TestGuru
